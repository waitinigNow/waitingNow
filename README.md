## 📣 프로젝트 소개
맛집에는 항상 웨이팅이 있었습니다. 하지만 기술의 발전으로 매장 앞 웨이팅 키오스크가 생겼고, 이제 더 이상 줄 서지 않아도 됩니다.<br>

웨이팅 나우로 고객은 **편하게 웨이팅**하고, 사장님은 고객을 **편하게 관리**해보세요.
